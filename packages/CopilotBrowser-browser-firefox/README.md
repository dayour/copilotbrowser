# @copilotbrowser/browser-firefox

This package automatically installs [Firefox](https://www.mozilla.org/firefox/) browser for [copilotbrowser](http://github.com/dayour/copilotbrowser) library. If you want to write end-to-end tests, we recommend [@copilotbrowser/test](https://dayour.github.io/copilotbrowser/docs/intro).
