# @copilotbrowser/browser-chromium

This package automatically installs [Chromium](https://www.chromium.org/) browser for [copilotbrowser](http://github.com/dayour/copilotbrowser) library. If you want to write end-to-end tests, we recommend [@copilotbrowser/test](https://dayour.github.io/copilotbrowser/docs/intro).
