# class: copilotbrowserException
> *Added in: v1.10*
>
> **Languages:** Java
* extends: `RuntimeException`

copilotbrowserException is thrown whenever certain operations are terminated abnormally, e.g.
browser closes while **Page.evaluate()** is running. All copilotbrowser exceptions
inherit from this class.
